# Sprints

Timestamped phased plans for implementation live here. They complement `AGENTS/BUILD_NOW.md` (current slice) with **sequencing, exit criteria, and maintenance notes**.

**Start here:** open the latest `20*-*.md` file by date in this directory.

When a phase completes, add a short “Done / deferred / changed” appendix to that plan file (or link to a follow-up sprint doc) so future readers know what drifted.

See also: `docs/adr/` for decisions that outlive a single sprint.
