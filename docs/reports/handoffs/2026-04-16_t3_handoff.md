## Current product shape

Loopgate in this repo is intentionally narrow:

- local HTTP control plane on a Unix domain socket
- signed policy and request-integrity enforcement
- approval and denial workflows
- append-only local audit ledger
- Claude Code hook governance
- governed MCP broker execution
- host-folder and sandbox mediation

This repo is not the place to reintroduce:

- model mediation or stored model connection flows
- speculative desktop UI product work
- continuity/memory work from sibling repos
- remote or enterprise assumptions in the local control path
- legacy product naming or retired surfaces
